# OpenCAD Responder - Web/PWA Version

This is the React + Tailwind version of the Mobile Responder application. 
It is designed to be deployed to **Vercel** as a fast, cross-platform Progressive Web App (PWA).

## Deploying to Vercel

1. Push this folder (`web-export`) to a new repository on GitHub.
2. Go to [Vercel.com](https://vercel.com) and click **Add New Project**.
3. Import your GitHub repository.
4. Vercel will automatically detect that this is a **Vite** project.
5. Click **Deploy**.

Within 1 minute, you will have a live URL. You can open this URL on any device (Desktop, iOS Safari, Android Chrome). On mobile devices, you can use the "Add to Home Screen" option to install it as an app.

## Running Locally

If you want to test it locally before deploying:

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.
